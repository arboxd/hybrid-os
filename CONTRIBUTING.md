# Contributing

## Branch Strategy

- main
- develop
- feature/*
- hotfix/*
- release/*

## Commit Convention

Examples:

feat: add workout dashboard

fix: resolve login issue

docs: update roadmap

refactor: improve folder structure

style: format source code

test: add unit tests

chore: update dependencies

## Pull Requests

- One feature per PR
- Keep PRs focused
- Update documentation if required
- Ensure project builds successfully
